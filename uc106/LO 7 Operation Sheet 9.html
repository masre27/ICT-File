Operation Sheet 9

Working with Columns

Note: The columns will only be shown correctly in Page Layout View. If you have Normal view the text will be shown as one long column.

Selecting required text

If you leave your cursor flashing in the text the column setup will apply to the whole document (or section).

If you select a portion of text Word will automatically insert a section break before and after the highlighted text so that only the highlighted text is placed in columns.

Changing the number of columns

There are two ways to change the number of columns.

  1. Click on the button, hold down the left mouse button and drag across the number of columns you require. Release the mouse button when ready.

  2. Select Format, Columns You can then move to the Presets area and click on the number of columns you wish to see. If you wish to have more than three equal columns you can use the number of columns option to type in the required number.

    Unequal widths

    Use Format, Columns. If you require two columns of unequal widths you can select the presets Left or Right. If these do not meet your requirements then select the number of columns you require and then move to the bottom of the dialogue box.

    The first width and spacing options are black and therefore available to change. If you change these all the columns will change to the measurements given in this first row.

    To make columns of unequal width you must first click in the Equal Column Width box to remove the tick. Then it will be possible to change all the measurements. Click the OK button when you are happy.

    Once you have removed the tick from the Equal column width option you can also change widths in Page Layout view by using the ruler.\

    Move the cursor to the on the right of the relevant column and drag to the right to increase the width, or the left to decrease.

    If you cannot see the on the ruler between the columns it implies that the Equal column width option still has a tick.

    Fine Tuning Columns

Space between columns

You can adjust the amount of space that Word leaves between right margin of text in one column

and the left margin of text in the next column.

To do this select Format, Column. You will see options shown right. You can now define the amount of blank space which you want between each column.

To change spacing between columns in Page Layout view you can also use the ruler.

The ruler below shows the ruler where columns are equal (no symbols)

To alter the space between columns move to the line between the white and dark grey on the ruler, hold the left mouse button down and drag towards the dark grey area to decrease the spacing between columns, (decreasing the column width). Drag away from the grey area to increase the space between columns. This will change the spacing between all columns.

When the columns are of unequal width (the ruler has hashes) you can change the spacing between each pair of columns

Line between

If you wish to see a vertical line drawn between each column click on the option at the right of the Format, Column dialogue box.

Making sure columns are of equal length (Balancing)

To make sure the columns are of equal length you must have a continuous section break at the end of the column area. Once this is inserted the text will automatically adjust itself so that the columns are of equal length. If this is not happening move into the following section and use File, Page Setup, Layout to check that the section start is continuous.
