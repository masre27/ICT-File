Word-Wrap and Capitals, Punctuation and coloured underlining

Word wrap

Word for Windows uses "word-wrap". This means that you do not need to press  at the end of a line. Word knows when the line is too long and starts a new line automatically. You will need to tell Word when you have reached the end of a paragraph by pressing .

Capitals

Single Capital Letter

Hold down the Shift key (at the bottom left or right of the main keyboard) and tap the required letter

Many Capitals

Tap the Caps Lock button (left of keyboard). Note that a light on the right of the keyboard is now on. Any letters you type will be capitals. To stop typing capitals tap the Caps Lock button again. (The light on the right of the keyboard will go off)

(note that pressing the Shift key with any of the number keys will give you the symbol shown at the top of the key)

Punctuation

It is normal to follow a full stop (.) or any punctuation mark containing a full stop (e.g. ! ? etc) with two spaces. A comma should be followed by one space. None of these punctuation marks should have spaces before them.

In general there should be two spaces between sentences and one within sentences.

If using brackets i.e. ( ) , [], {} there should be a space before the open bracket ( but none after. There should be no space before a close bracket ) but one after. (Like this) for example.

Coloured Underlining

Red wavy underlining under a word indicates that the word is not recognised. It may be spelt incorrectly or just not be in Word’s dictionary.

Green wavy underlining under a word or phrase indicates that Word thinks there is something wrong with the grammar. This may be anything from incorrect spacing to incorrect wording. Word’s suggestions are not always right!

Blue wavy underlining indicates inconsistent formatting.

Purple dotted underlining implies presence of a smart tag which is a shortcut for doing something in another application (usually Outlook – and American based)

Insert/ Overtype

Normally text which you type is inserted at the position of the cursor.

Press the Insert key to alter this so that what you type will overtype any text which

already exists after the cursor position. Change back to insert mode by pressing the Insert key again. To see how Word is set up, look at the bar at the bottom of the window

Notice the section which says OVR. When this is grey then you will insert text. When OVR shows in black you will overtype existing text.

You can also double click the OVR option to switch it on or off, or use the menus by selecting Tools, Options and click on the Edit tab. Then click the Overtype mode option to switch it on or off.

Different ways to open documents

In the File,Open dialogue box select the drive filename etc and click on the down arrow at the right of the button.

Open: the normal option

Open Read-Only: no changes can be saved to the original file.

Open as Copy: creates a copy with the words Copy(n) of before the original name (where n is a number)

Open in Browser: this is only available if you are opening a web page

Open and Repair: will attempt to repair a faulty file

Changing Documents

To see what documents you currently have open click on the Windows menu item

You will see the Arrange All option, and below that a list of all files which you currently have open. The current file has a tick by it. To change to a different document click on its name and it will be brought to the front of the screen

Click on the down arrows to see other options. The menu will now expand to show other options


New Window will create a new window for the current document - so you can compare text in different parts of the document, or copy text between different parts of the same document.

Use Arrange All to show all the documents at the same time.

Split will allow you to divide the window into panes so that you can view more than one part of the document at a time. Once you have a split in place the menu will show Remove split

Note: The more files you have open the more memory is being used.
