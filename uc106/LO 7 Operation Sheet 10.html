Bullets and Numbering / Style

Inserting Numbering/Bullets

To automatically number your paragraphs highlight the paragraphs concerned and click on the button. To add bullets highlight the paragraphs concerned and click on the button. Whichever button you have clicked will have a blue box around it.

You can also apply the numbering by simply typing “1.” And press the tab key followed by your text. When you press the Enter key the next number will automatically be inserted.

Note: If you wish to create a new line within a paragraph without adding a number use the Shift Enter keys to start a new line.

Removing Numbering/Bullets

You can simply press the Enter key twice, or press the Backspace key to delete the last number.

You can also click on or at the end of a numbered or bulleted list.

Note: If you use numbering again later it will automatically start from where the numbering left off last time.

Adjusting numbering start point

Restarting numbering

To start a new list click with the right mouse button on the number at the beginning of the relevant line and select from the shortcut menu

Continuing numbering after a break

Click with the right mouse button on the number at the beginning of the relevant line and select from the shortcut menu

Altering style of the Bullets/numbering

Select Format, Bullets and Numbering. Make sure the Bulleted or the Numbered tab, as relevant, is at the front. You can choose from six different bullet symbols/styles of numbering. Simply click on the picture of the option you want to use.

Note: There is also an option for outline numbering. This is only useful if you have set up your document using the Heading styles for your headings. (This is covered in detail in a later course)

Changing Bullets

Select Format, Bullets and Numbering, Bulleted, click on the closest format to what you want and click on the Customize button.

You can select a different symbol for the bullet point by clicking on one of those shown, or by clicking on Character and selecting a symbol from the full lists there. Wingdings is a useful font to look for symbols in. You can also click on Picture to select a coloured picture.

Changing numbering

Select Format, Bullets and Numbering, Numbered and click on the Customize button to allow you to alter the way the numbering looks:

In the first box you can type the symbols which you want to see before and after the actual number.

e.g. in the example on the left you could delete the brackets and type a full stop after the 1, or you could type item before the number.

Click the button to change the font style and size etc. for the numbers only.

Number style allows you to choose whether to use words, letters, arabic or roman numbers etc.

Start at allows you to define a starting number other than starting at 1 – for example if you have broken the list and now wish to continue it.

Changing the indents for numbering

Select Format, Bullets and Numbering, Numbered

Number position: This adjusts the position of the number in the space at the left. (Though I see no logic in it!)

Aligned at: sets the distance from the margin to the number

Tab space after: sets the distance from the margin to

the start of text following the number

Indent at: sets the amount of space between the margin and any text which continues onto a 2nd line (i.e. a hanging indent)

Changing the indents for bullets

Bullet position: Indent at: sets the distance of the bullet from the left margin.

Tab space after: sets the distance from the margin to the start of text following the number

Indent at: sets the amount of space between the margin and any text which continues onto a 2nd line (i.e. a hanging indent)

Introducing Styles

A Style is a collection of formatting. If you use a particular collection of formatting repeatedly you can save it as a style and apply it to text with a click when required. Styles are saved by default as part of the file.

Creating a style

To create a style, format the paragraph or text in the required way, click in the style box on the left of the tool bar

Type a name for the style and press Enter

Applying the style

To apply the style to a paragraph, move the cursor into the paragraph, click on the arrow at the right of the box, click the required style name to apply it. Note that the name of the style in the list is shown using the formatting of the style.

Manually changing paragraph formatting

If you make changes to formatting of a paragraph which is based on a style you will create a variation on the original style which will be listed in the styles.

E.g. I have a heading based on the style Heading 2. If I highlight the text and make it italic I will then have two styles listed, as shown on the right

Returning the formatting of a paragraph to that of the basic style

Highlight the paragraph and press Ctrl spac


Editing A Style

To edit a style select Format, Styles And Formatting. A menu will appear to the right of the screen. The styles are listed under Pick formatting to apply. The current style will be shown with a blue box around it.

To edit a style without applying it to the current paragraph move the cursor over the name of the style at the top of the Style window. A down arrow will appear at the right of the box.

, Click on the down arrow and select Modify.

(You can also right click on the relevant name in the list below and select Modify).

Editing formatting

Use the formatting tool buttons or click and select the area of formatting you wish to change. Make the change. Repeat for all changes.


Add to template

Click this option to ensure that the style will be all files based on the same template that you are currently using and not just the current one.


Shortcut keystrokes

Click the Format button, click in the Press new shortcut key box

Press the key sequence you would like to use (including Alt or Ctrl or both), check whether it is already assigned. If it is unassigned click the Assign button. Once you have made any necessary alterations you should click on the OK button.


When you come back to the first Style dialogue box you will see that you have two buttons at the bottom right to choose from.


If you click the button you will apply the selected style to the paragraph(s) in which your cursor is.

You may prefer to click on which saves the changes you have made without applying the style.

Note: The close button will only appear once you have made changes to a style, otherwise you will see the button. Deleting Styles

To Delete a style

Make sure the styles are showing on the right (Format, Styles and Formating)

Right click on the relevant style and select Delete.

The delete option will be unavailable if the style you have highlighted is a standard style. You can only delete styles which you have created.













Checking on the usage of a style

You may like to check whether a style is being used before you delete it

Right click on the relevant style. Note the option Select all n instances. This indicates that n paragraphs are using that style. Click on this option and scroll though the document to see the highlighted paragraphs. These are paragraphs which use the style.
