Working with Headers/Footers

Headers / Footers

Headers and footers consist of text which you want to appear at the top or bottom of all, or a selection of pages. You will find it easiest to ensure you are in normal mode before setting up a header/footer.

Distance from the edge of the page

To define the distance of the Header/Footer from the edge of the page select File, Page setup and click on the Layout tab.

In the From Edge section you will see the options on the right. The Header option defines the distance from the top of the page to the top of the header text. The measurement should be less than that of the top margin.

Similarly, Footer defines the distance from the bottom of the page to the bottom of the footer text and should be less than the Bottom margin.

You can either type in a measurement or use the arrows.

Note: Some Hewlett Packard Deskjet printers can only print footers at a minimum distance of 2.1cm (0.8”) from the bottom of the page.

Adding Page Numbers

Select Insert, Page numbers

Position: This allows you to select whether you want the page number to appear at the top or the bottom of the page

Alignment: Select whether the number should be centred, on the left of each page, on the right of each page, on the binding side (inside) of each page or the outside of each page

Show number on First Page. If you do not wish the number to show on the first page then remove the cross from this option. Note that the first page will still be considered to be page 1 and numbering will start from 2

Unlinking the Headers And Footers

(The explanation following deals with headers. The situation for footers is identical)

When text is typed in a header in one section it is copied by default to all other headers in the document. You may want some sections to have headers and some not. To allow this

you must unlink the headers.

When you first look at a header (using View, Header and Footer) you will see the words Same as Previous shown at the top right of the box.

This means that anything you type in this header is copied to the previous header, and anything typed in the previous header will be copied to this header. By default all headers in the document are linked in this way. If you have odd page and even page headers all the odd page headers are linked together, and all the even page headers are linked together, likewise all first page headers are linked.

Breaking the link

View the header at the point where you want to have a different header and click the button so that it comes up and looks like . The words Same as previous should vanish from the top right of the box. (To recreate the link just click on the button again)

You may need to clear any unwanted text from headers, but anything you type in a header prior to the place where you broke the link will not be copied to any headers which occur after it (and vice versa). Note that any headers following the unlinked header may still be linked to it and therefore have its text copied to them.

Adjusting Page Numbers

To alter the way page numbers work you will need move to the relevant section and select Insert, Page numbers. Then click on the Format button.

Number Format

Select the particular format the numbering should take (roman numerals, letters etc.)

You can choose to include the Chapter number if you have used the standard styles Heading 1 - 9 for the Chapter headings

Starting number

You can define that the numbering continues from the previous section, or that it will start from a particular number in that section.

The page numbering will be inserted in the header or footer which you can then view in the usual way

Once you have clicked on the OK button from this window you will return to the previous window.
