Operation Sheet 7

Searching for Text

To search, select Edit, Find. In the Find What field type the word for which you wish to search.

To search for the next occurrence of the search text click on Find next.

Note. If you have text selected Word will initially search the selected text only

To do more complicated searches, such as for text formatted in a particular way, or for a particular style, click on the button.

Select Match Case to find only text that is identical in case to the text which you typed. Otherwise case will be disregarded e.g. if you type hand Word will not find HAND

Select Find whole words only to prevent a search for turn finding the words return, returned, outturn, turning etc.

Use wildcards allows you to look for words that begin with certain letters etc. You must have this selected to use pattern matching commands. Switch it off when not required

You can see pattern matching controls available by clicking on the Special button. e.g.

h*t finds heat, heather, heart etc. (* represents any number of characters – including spaces)

h??t finds heat, heather, but not heart (? represents one character)

h[a-e]t finds hat, het, hit but not hit or hut ([n-n] gives an allowed range)

h[!a]t finds anything except hat ([!n] excludes the letter defined)

h[!ao]t finds anything except hat and hut including ‘h t’. (all letters following! are excluded)

h[! ]t finds any three letter word (the space is excluded)

<h finds h where it begins a word

h> finds h where it ends a word

^p finds an end of paragraph mark

Sounds Like (English) allows you to search for words which sound like the phrase you type.

Find all word forms allows you to search for words which have some form of relationship: eg different parts of verbs (think, thought) etc

In the Search box state whether how you wish to search the document:
Down from your current position to end of the document
Up from your current position to the beginning of the document
All throughout the whole document

To search for paragraph marks etc., click on the button followed by the button and select the character for which you want to search.

Highlight all items found in: click this so that all occurrences of the word are highlighted. You can then scan through the document. As soon as you click the highlighting will disappear.

Replacing Text

For example if you have used the wrong word throughout a document

Select Edit, Replace. The following dialogue box will appear:







(All options are the same as for Edit, Find)

In the Find What field type the text you wish to replace

In the Replace with field type the text you wish to replace it with

Either choose Replace All so that all occurrences of the text are replaced automatically in one pass, or confirm each replacement by using the Find Next button to find the first occurrence. Click the Replace button to make the replacement and move to the next occurrence , or Find Next to skip to the next occurrence.

If you have a document in which a  has been pressed at the end of every line you can remove the end-of-paragraph mark ( ) by typing ^p in the first field and leaving the second blank.

Other special characters are: ^t=tab, ^n=new line, ^c= column break, ^d=section break

Note: These are case sensitive

Replacing Formatting

Select Edit, Replace

Move to the Find box and type the text you wish to replace. If you only intend to replace formatting and not text the Find What box may be left blank. Click on the Format button at the bottom of the dialogue box to define the formatting for which you are searching.

Move to the Replace with box and type the text you wish to change to. Again, you may not actually be changing any text, just its formatting, in which case the Replace with box may be left blank. Click on the Format button at the bottom to define the formatting which you wish to change to.

Note that any formatting is listed underneath the relevant box

You must remember to click on the No formatting button at the bottom of the dialogue box once you have finished dealing with that formatting.

In this example the word stanley will be highlighted and aligned left wherever it is found

Notes:

If you replace <t* with italic formatting only the inital t will be italicised and not the whole word. (remember that if you do not pattern matching selected Word will actually look for the characters <t*)

If you replace <x??? with , say, bolding then an initial x and three following characters will be made bold whether they make up a whole word or not.

AutoCorrect

AutoCorrect automatically corrects typing errors as the text is entered. It can also be used to insert frequently used phrases. Some common errors are set up already, e.g. if you type teh and press the spacebar the word will be changed to the. The correction is triggered by the spacebar, or punctuation.

Adding new Auto corrections

To set up your own Auto Corrections select Tools, AutoCorrect Options. The dialogue box below appears.

  1. Type the incorrectly spelt word in the Replace box

  2. Enter the correct word in the With box

  3. Click on the Add button.

To Remove existing Auto corrections

Highlight the correction in the list and click the Delete button

Turning off the Autocorrect feature

Select Tools, Autocorrect Options and remove the tick from the Replace text as you type option

Note: There are several other options for AutoCorrect.

Correct TWo INitial Capitals: When two initial capitals are typed in error, as soon as the space bar is pressed the second capital will be changed to lower case.

Capitalise first letter of sentences: This will capitalise the first letter of any word following a full stop

Capitalize first letter of table cells: This will capitalise the first letter of any wod in a table cell.

Capitalize names of days: This will capitalise the any days typed i.e. Wednesday.

Correct accidental usage of cAPS LOCK key: This will automatically change things like “tHIS” to “This”.

There are also useful options under the tab Autoformat as you type.

Change “straight quotes” to “smart quotes”: When you type the normal quote marks found above the 2 at the beginning and end of text e.g. "like this", they will be changed to the relevant shaped quotes e.g. “like this”.

Ordinals (1st) with superscript: This automatically changes 1st to 1st etc.

Fractions (1/2) with fraction character (½). This only applies to the fractions ¼, ½ and ¾.

Creation of entry:

Type the text you wish to save (including formatting) and highlight it (If you have included paragraph formatting ensure you have included the symbol in your highlight)

Select Insert, AutoText, New.

Type a name in the box and click on the OK button.

If the name is 5 characters or more the “auto complete” option will work when you have typed the 4th character as shown below.

Note The AutoText is normally saved in the Normal template – which means that it will be available to all documents.

Insertion of entry into text:

Either Type the name of the entry and press the F3 key

or Select Insert, AutoText, AutoText, select the entry you want to insert and click on .

Editing an existing piece of AutoText

Insert the AutoText entry which you wish to edit into a document and make the necessary changes.

Highlight the text and select Insert, AutoText, New. Type the previously used name and click OK. You will be asked to confirm that you want to redefine the AutoText entry. Click Yes to accept this.

Deletion of existing AutoText

Select Insert, AutoText, AutoText and highlight the name of the AutoText which you wish to delete. Click and then click the OK button.

Printing a list of AutoText entries

Select File, Print and choose Auto text entries under the Print What option.

Sections

A Section is a part of the document. You can use sections to set up different layouts for different parts of your document. You indicate the end of a section by selecting Insert

particular Section break you want and click OK.

A line of the type below indicates the end of a section.

Next Page means that a new page will be started where you insert the section break.

Continuous means that following the break the text will continue on the same page.

Even page and Odd page: In these cases a new page will start after the section break and the text will be placed on the next even or odd page as defined.

Always ensure your cursor is inside the section you are trying to change.

What section am I in?

Look at the Status bar (at the bottom of the window) to see which page and which section you are in

Note: You can also see which page of the total number you are on.

Changing Section Break Start

To change the start of the section select File, Page Setup, and the Layout tab. You will see the option and you can click on the down arrow to change the start style.

Deleting Section Breaks

DO YOU REALLY WANT TO? All the formatting for a section is stored in the section break symbol at the end of the section. So if you delete a section break you will also lose the formatting for that section. The section will take on the formatting from the next section break it finds.

If, knowing this you really do want to delete the section break then simply place your cursor on the section break symbol and press the Delete key on the keyboard.
