Setting a criterion

To choose a specific group of records to use you can set a criterion.

For example, if I wish to write only to people who live in the town of Gingerville I would set the criterion as follows:

C lick the down arrow to the left of Town

A list of all the data is shown. Click on Gingerville

The list will close and the down arrow will change to blue to remind you that a criterion has been set on this field.

Y ou will now only see the records which meet the criterion

When you close the Mail Merge Recipients box and view the data on screen, or merge the data to a printer, only these records will be used.

Using more than one criterion

You can set criteria for more than one heading. If you do so then all criteria must be met. There is, however, and alternative way to deal with this, which gives a little more flexibility.

Click on the down arrow next to one of the headings and click Advanced

Under Field select the field name which you wish to use to define data

Under Comparison select the option you wish to use

Under Compare to type the text, number or date which you wish to match

Repeat this for as many criteria as you wish to set up. If you use And then both criteria must be met. If you use OR then either one of the criteria may be met

e.g.

Title = Professor AND Town= Gingerville will select only those people whose title is Professor who live in Gingerville,

Title = Professor AND Town= Gingerville will select those people whose title is Professor and those people who live in Gingerville (Effectively two groups of people)

The Address Block and Greeting line

At Step 4 of the Mail Merge Wizard we have the opportunity to add fields. Previously we chose More items but there are two other options which may save us time if they are set up correctly.

The Address Block

This will enable us to add the address of the recipient in one go – that is, without having to add 5 or 6 different fields. However, it will only work correctly if you have field names which Word can match with its expectations.

T he Address Block expects the following fields:
Last Name, First name, Courtesy Title, Company, Address 1, City, State, Postal Code and
Spouse First name

If MS Word can recognize those pieces of information in your column headings it will. E.g. it recognizes Title as Courtesy title and Surname as Last Name etc.

If MS Word cannot recognize the fields it will assume they do not exist and work without them. It is possible, however, that the information exists, but not with recognizable field names.

In this case you can advise MS Word which of your field names match its expected ones.

To do this, click and then click (You can also click on the toolbar)

For each pieces of information shown on the left, click in the box on the right and select the matching field

Once this has been done you can use either the Address Block option, or the Greeting line with impunity.

T he Greetings Line

Select the options from each box that you wish to see.

Note that there is an option for invalid recipient names. I.e if there is no Title and Surname MS Word will enter a general greeting instead of the line being left blank.

Editing the Data source from within Main Document

Click on the button on the Mail Merge toolbar:

If the source file is a Word file you will se a form and can use this to enter or edit data. The data source file will be changed – but you must remember to save it!

There will be no apparent difference in the way you access the data – except that you cannot edit the data source. You must go back to Excel to add data for example.
