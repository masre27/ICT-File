Sections

Creating sections

A Section is a part of the document. You can use Sections to set up different layouts for different parts of your document. You indicate the end of a section by selecting Insert Break.

You can then select the particular Section break you require and click OK.

The end of a section is indicated by a line of the type below.

Next Page: a new page will be started after the break.


Continuous: text will follow the break on the same page.

Even page and Odd page: a new even or odd page will start after the break.

Changing Section Break Start

Always ensure your cursor is inside the section you are trying to change.

Click on the downwards arrow to change the start.

Which section am I in?

If you look at the bottom of the screen you will see that there is a line (status bar) which shows what page number you are looking at. To the right of the page number it shows what section – Sec you are in. To the right of that you can see how many pages in total there are and which page you are currently viewing.

Note that in this case we are looking at page 19 which is in Section 3. The confusing thing is that there are 28 pages and we are on the 23th page. This is because in this document the first section is not numbered so there are 4 un-numbered pages before the page numbering starts. Thus the first number refers to the number of the page while the last set of numbers refers to the position of the page within the document as a whole.

Margins

You can alter the margins (i.e. the distance from the edge of the paper to the text) by clicking on the up and down arrows, or by typing a measurement in the box.

The Gutter margin is an additional margin to allow for any binding you may be using on the finished document.

Orientation



This prints text on the page so that the narrow edge of the paper is at the top (as with a portrait picture)



The text will be printed so that the long edge of the paper is at the top.

























Pages

Normal: is the usual situation.

Mirror margins: the margins are reversed on facing pages. The margins will change from Right and left to inside and outside.

2 pages per sheet: divides each piece of paper into two pages. If you apply it to an A4 document with one page you will see two A5 pages, each with the page margins set as defined above.

Book fold: This will automatically set the orientation as Landscape and will print two pages on each piece of paper.

Changing Paper Size

Select File, Page Setup, and click on the Paper tab.

Click on the down arrow on Paper Size and choose a size for the document, or part of the document (if you have set up sections).

The usual paper size is A4











Note:

  1. If you have made changes which you wish to be used for all future documents you can click on and this will make all the changes on all the tabs in this dialogue box.

  2. On the Paper tab under Paper Source you can select which tray of the printer (if it has more than one tray) will be used for the first page, and which for other pages. (e.g headed paper for first page, plain for rest).
